# Mockups & marketing assets

Standalone HTML design mockups and brand/marketing asset generators. Each file is
self-contained — **open it directly in a browser**, no build step required.

## Files

- **`linkedin-assets.html`** — LinkedIn & marketing brand-asset generator. Renders
  brand assets to `<canvas>` at exact LinkedIn pixel sizes so you can **right-click →
  "Save image as…"** (or use the Download buttons) to get spec-sized PNGs:
  - Company page logo — **300 × 300**
  - Company page banner / cover — **1128 × 191**
  - Personal profile banner — **1584 × 396**
  - A range of logo variations (colour-flipped, bright, and large `(R)` monogram
    options across solid / gradient / split / radial backgrounds), each with a
    small white-chip preview to judge legibility at LinkedIn thumbnail size.

- **`first-fold.html`** — Homepage hero "first fold" concept explorations (the
  country-presence list / "spinner" variations that fed the live homepage hero).

- **`homepage.html`** — Full homepage design mockup.

- **`about-us.html`** — About page design mockup.

- **`about-leadership-concepts.html`** — Leadership section concept explorations.

- **`bios-styles.html`** — Team / leadership bio card style explorations.

## Notes

- Assets use the brand palette from `src/theme.ts` (teal `#38C6BA`, gold `#C8A66E`,
  background `#14181D`) and the Inter typeface.
- These are reference/working files, kept in the repo in case the assets are needed
  again — they are not part of the built site (`src/`).
